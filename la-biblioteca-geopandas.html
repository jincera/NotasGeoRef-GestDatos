Notas para la sección de Sistemas Geo-referenciados

Capítulo 2 La biblioteca GeoPandas

GeoPandas es una extensión de la biblioteca Pandas que permite trabajar con datos geoespaciales de una manera muy eficiente. Su estrucutra de datos central es GeoDataFrame, una subclase del DataFrame de pandas que puede incluir columnas llamadas geometrías o GeoSeries en las que se almacena la información geoespacial, y define una serie de operaciones espaciales. De esta manera, en un GeoDataFrame se pueden combinar datos tradicionales con geometrías (puntos, polígonos, líneas) y hacer operaciones sobre ellos.

A diferencia de la mayoría de paquetes GIS, GeoPandas permite que un mismo GeoDataFrame se especifiquen varias geometrías. Cada geometría puede tener su propio CRS y solo una geometría puede estar activa. El poder contar con varias geometrías es muy útil cuando es necesrio trabajar en diferentes proyecciones. Además, permite cambiar geometrías a distintos CRS con el método to_crs().

GeoPandas soporta la mayoría de los formatos de archivos, como Shapefiles (.shp), GeoJSON y KML. Los archivos se leen con el método read_file() y se pueden guardar GeoDataFrames como archivos geoespaciales con el método to_file().

En cuanto a la visualización, GeoPandas integra su funcionalidad con matplotlib y otras herramientas, lo que permite generar rápidamente mapas y visualizaciones geoespaciales con el método plot().

2.1 Operaciones geoespaciales.

GeoPandas ofrece una gama muy amplia de métodos para realizar operaciones espaciales, incluyendo la agrupación o la intersección de datos en distintas capas (en distintos GeoDataFrames) en función de su relación espacial. Entre las más importantes, se tienen:

  • Métodos Geométricos. Permiten realizar operaciones sobre las geometrías de los datos geoespaciales (puntos, líneas y polígonos).

    • area: Calcula el área de cada geometría en el GeoDataFrame.
    gdf['area'] = gdf.geometry.area
    • centroid: Calcula el centroide de cada geometría.
    gdf['centroid'] = gdf.geometry.centroid
    • length: Calcula la longitud de líneas o perímetros de polígonos.
    gdf['longitud'] = gdf.geometry.length
    • distance(otro): Calcula la distancia mínima entre cada geometría y otra geometría.
    gdf['distancia'] = gdf.geometry.distance(otra_geometria)

  • Métodos de Relaciones Espaciales. Estos métodos permiten evaluar relaciones espaciales entre geometrías.

    • intersects(otra): Devuelve True si la geometría intersecta con otra geometría.

    • within(otra): Devuelve True si la geometría está completamente dentro de otra geometría.

    • contains(otra): Devuelve True si la geometría contiene completamente otra geometría.

    • overlaps(otra): Devuelve True si las geometrías se superponen parcialmente.

    • disjoint(other): Devuelve True si las geometrías no tienen intersección.

  • Métodos de Operaciones Espaciales. Estas operaciones permiten modificar o combinar geometrías.

    • sjoin(): Realiza una unión espacial entre dos GeoDataFrames, combinando datos basados en relaciones espaciales.

    • overlay(): Realiza operaciones espaciales como intersección, unión o diferencia entre dos GeoDataFrames.

    • clip(): Recorta (clip) las geometrías de un GeoDataFrame usando los límites de otro.

    • dissolve(by): Agrupa geometrías basadas en una columna y disuelve sus límites. Es vagamente similar a merge para DataFrames ```
